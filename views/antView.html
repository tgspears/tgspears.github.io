At this point the game can handle up to 1378 moves, at which point it hits the edge of the game board and breaks.

As time goes on I will try to add features to this 'game' to try to turn it into an actual game.

Some potential features are:

  • Animated 'play'
  • Dynamically expanding board
  • Pattern recognition (given time, the game stabilizes in an infinite pattern)
  • Customizable game board
  • Puzzle boards:
    • Given a beginning point, customize the game board so that the ant hits a specific end point
    • Given a particular pre-populated game board, set the beginning point so that the ant hits a specific ending point
  • And more things in mind that are potentially too ambitious for me to even mention!
